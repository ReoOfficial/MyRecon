# Reports Directory

The `reports` directory stores JSON reports generated by MyRecon.

After a scan, the default report is saved as:

```text
reports/recon_report.json
```

The report contains structured information collected from all targets scanned during the current application run.

Example:

```json
{
    "target_count": 1,
    "results": [
        {
            "input_target": "example.com",
            "success": true,
            "status_code": 200
        }
    ]
}
```

Generated JSON reports are excluded from Git using `.gitignore`.

The `README.md` file is kept in this directory so that the purpose of the folder remains visible in the Git repository even when no generated reports are committed.
